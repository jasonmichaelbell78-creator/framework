    - Sections: Overview, Phases/Steps, Dependencies, Success Criteria, Version History
    - Includes dependency graph placeholder

    **REFERENCE.template.md**: For reference/companion documents.
    - Sections: Overview, Quick Reference, Detailed Reference, Examples, Related Documents
    - Designed for lookup, not linear reading

    **GUIDE.template.md**: For how-to guides.
    - Sections: Overview, Prerequisites, Steps, Troubleshooting, Related Guides
    - Step-by-step instructional format

    **JSONL_SCHEMA.template.md**: For documenting JSONL file schemas.
    - Sections: Overview, Fields (table: field, type, required, description), Example Record, Validation, Related Schemas
    - References base-record.schema.ts as the universal base

    **PROJECT_GENESIS.template.md**: For project genesis documents.
    - Sections: Project Identity, Core Value, Key Decisions, Architecture Overview, Technology Stack, Development Workflow, Version History
